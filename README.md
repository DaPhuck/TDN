# 📊 Group Midterm Test – Data Analysis Project

## 📌 Overview
This repository contains a **group-marked midterm test** completed using **Jupyter Notebook**.  
The project focuses on applying fundamental data science techniques to analyse and interpret a dataset.

---

## 🎯 Objectives
The main objectives of this assessment were to:

- Perform structured **data analysis**
- Extract meaningful **attributes (features)** from raw data
- Create clear and informative **data visualisations**
- Interpret findings based on analytical results

---

## 🔎 Project Components

### 1️⃣ Data Analysis
- Exploring the dataset
- Generating summary statistics
- Identifying patterns, trends, and relationships

### 3️⃣ Data Visualisation
- Creating graphs and charts to represent insights
- Comparing variables visually

---

## 🛠️ Tools & Technologies
- Python 3
- Jupyter Notebook
- Pandas
- Matplotlib
- Seaborn
